# AI Artifacts

This directory contains chat exports, prompts, and AI collaboration artifacts from the development of this Claims Processing System.

## Required Contents

Per the assignment requirements, include:

1. **Chat exports** - Full session transcripts from AI tool usage
2. **Notable prompts** - Key prompts that drove significant implementation decisions
3. **AI corrections** - Instances where you caught and corrected AI-generated code

## Format

Exports can be in:
- Markdown (`.md`)
- JSON (`.json`)
- Plain text (`.txt`)

## Example Structure

```
ai-artifacts/
├── session-2026-04-28-domain-modeling.md
├── session-2026-04-28-adjudication-engine.md
├── notable-prompts.md
└── corrections-made.md
```

## Assignment Requirement

**Required for submission.** Without AI artifacts, the assignment cannot be evaluated as the process is part of the assessment.
